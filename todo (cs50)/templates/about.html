About the Kawaii To-Do

Welcome to Kawaii To-Do!

This is my final project of CS50X, where I applied what I've learned about web development using various technologies like HTML, CSS, JS, Flask, and SQLite.

The app provides an intuitive way to manage your daily tasks. Whether it's personal goals, work-related items, or anything in between, stay organized and achieve more!

Key Features

  • Create, Update, and Delete Tasks: Effortlessly manage your tasks by adding, updating, or removing them once completed. 🌱
  • View Your Todo List: Quickly access and review your entire list of tasks at any time. 📋
  • Responsive Design: Access your to-do list from any device—be it a phone, tablet, or desktop. 📱💻

About the Development

This app was built using Flask, a lightweight Python web framework, with a focus on simplicity and functionality. The stack includes:

  • HTML, CSS, and JavaScript for Frontend: These technologies create the user interface for the Kawaii To-Do app. HTML structures the layout, CSS gives it a clean and simple design, and JavaScript adds interactive elements. 🎨
  • Jinja Templating: Dynamically renders content based on the data passed by Flask, allowing for an interactive and personalized user experience. 🖥️
  • Flask for Backend: Manages the app's routes and server-side logic, handling requests and responses to ensure smooth functionality of the task management features. ⚙️
  • SQLite Database: Stores user data and tasks securely, ensuring your to-do list is preserved and easily accessible. 🔐
